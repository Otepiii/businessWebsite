Post a RFP

projectpartner allows you to make a request for proposal (RFP) that announces and provides details about a project, as well as solicits bids from contractors who will help complete the project.

To Start, follow these steps


  • Input Evaluation Process: (HINT Usually) Total Points Awarded & Cost

    •  Select if you want bids to include "Exclusions Omissions" or all works per division

         ie all electrical works indrawings and specs.


      Select if you want subtrades to quote with "Alternative Works" to the drawings or specs

         ie Brand name Tyveck is used in bid documents, but a alternative product that meets or

         exceeds spec is accepctable


  • Select all the submission criteria you wish to use to compare proponents
